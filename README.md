Welcome to Menda Pekwm Theme!
===========================

This theme has been ported theme Menda Official to suit users of Manager Pekwm windows

**Install**
-------------
Download file and unzip, move the folder Menda for `~/.pekwm/themes`

> **Note:**

Please report any error.
